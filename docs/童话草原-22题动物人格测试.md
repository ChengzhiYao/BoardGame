## 1. 算分与归一化规则（给代码 · `lib/meadow/persona.ts`）

1. **属性**：`最终 = 25（幼态基线）+ 物种修正 + Σ测试点 × 3`，clamp 5–80（幼崽弱，靠成长抬升）。
2. **本能**：`最终 = 物种修正 + Σ点`（约 0–20）→ 成长倍率 `mult = 1.0 + 最终/20`。该领域技能成长速度 = 基础 × mult × 年龄阶段。
3. **性格轴**：`轴值 = clamp(Σ正极 − Σ反极, −100..100) × 系数`，展示为倾向（如"凶猛 +40"）。
4. **食性**：`肉 = Σ食:肉`，`草 = Σ食:草`。`diet = 肉 ≥ 草+2 → carnivore；草 ≥ 肉+2 → herbivore；否则 omnivore`。食性同时给物种亲和加成（见 5）。
5. **物种亲和 → 出生物种**：`亲和[物] = Σ该物加分`，并把 **食:肉 总分加到三种肉食物种（狐/鼬/鸮）**、**食:草 总分加到草食物种**。
   `最终权重[物] = 基础稀有度[物] × (1 + 系数 × 亲和[物])`，归一化后**加权随机**掷物种。
   - 草食/杂食物种：系数 0.10，单物封顶 ×2.5。
   - **肉食物种（狐/鼬/鸮）特例**：系数 0.18、封顶 ×6——让"答得越凶猛、越像猎手，越可能生为肉食者"（极端凶猛作答可把肉食概率推高到 ~50%）。
6. **特性**：每特性累计 token，**≥3** 入候选，取最高 1~2 个。物种天生特性（**翅膀**=鸦/鸮、**尖刺**=猬、**掘洞**=獾/兔/鼬）由物种自动附带，不占名额。
7. **跳过**：该题不计分，按"随机抽取本题某一选项的权重"补入（命运替你掷）。
8. **出生地点/所属群体**：物种定后按其分布随机掷定（不受测试影响）。

> 判定法：d100，`掷骰 ≤ 相关属性 + 技能 − 难度` 成功（沿用 CoC 成功等级）。

## 4. 结果页 · 性格解读 + 物种揭晓（唯一一次 LLM 调用）

算分掷物种后，把 `scores`（属性/本能/性格/食性）+ 命中特性 + 掷定物种 + 3~5 条代表性作答喂给 `main` 模型，生成第二人称解读：
- "**你是怎样一只动物**"：综合性格与本能的速写。
- "**为什么这么判断**"：引用关键选择佐证（如"在受伤的猎物前你毫不犹豫——你的牙齿天生为追猎而生"）。
- "**你将作为【物种】降生**"：点出物种、食性、1~2 起始特性与最擅长的本能，并以童话绘本口吻欢迎它来到草原。
