## 0. 一句话定位

**「你是一本童话书里、一片小草原上的一只会说话的动物。从出生那天起，吃，或者被吃。」**

- 世界是**一本童话书里的一页草原**，所有动物都会说话、有名字、有自己的小社会——表面温暖治愈，底下却是**真实的食物链**（肉食 vs 草食）。这种"可爱的皮 + 残酷的里"正是它的钩子。
- 一座**岛大小**的封闭草原，**生态完整**：草木→草食动物→肉食动物，四季流转、昼夜更替、饥饿与捕食。
- 你随机出生为其中**一种动物**（由 22 题人格测试塑造），活出（并可能被吃掉）一生。死了才能重测、重生为新动物。

## 1. 与《列王纪》的关系（复用 vs 差异）

**直接沿用（不重写）：**
- **×10 真实时间**：真实 1 天 = 世界 10 天，连续时钟、离线也走（§列王纪 3）。
- **行动时间占用**：一次只做一件事，发起→倒计时占用→完成结算/被打断/主动放弃→恢复自由（§列王纪 5D）。觅食、潜行、迁徙、筑巢、捕猎都是时间占用行动。
- **情报差·感知层**：只知道你这只动物感官范围内的事；远处的危险靠"风里的气味、同伴的尖叫"间接传来（§列王纪 5B）。
- **永久死亡 + 一号一命**：每 Gmail 账号唯一动物；被吃/老死/饿死后才能重测重生（§列王纪 5A）。
- **人格测试出生**：开局做测试 → 确定性算分 → LLM 性格解读 → 随机+亲和加权掷定**物种**（§列王纪 5C，本文 §6）。
- **混合世界推进**：每日心跳推进生态（季节、种群、天气、NPC 动物行动）+ 玩家行动即时结算（§列王纪 3.2）。
- **机制确定性、叙事 AI**：捕食/觅食/逃逸用确定性 d100 判定，LLM 只负责叙事与关键事件。

**草原的差异 / 缩小：**
- **世界小**：一座岛大小、~6 个地点的封闭草原，而非三王国大陆。
- **物种少**：~10 种动物（含肉食/草食/杂食），而非 10 族 + 宗族。
- **测试 22 题**（列王纪是 24）。
- **核心循环更聚焦**：生存（吃/被吃/避险/繁衍/动物社会），而非权谋。这正是要先验证的"最小可玩闭环"。
- **童年期局部加速**沿用（幼崽很快长大），但动物寿命短，一生节奏更快、更适合验证。
- **计费**：MVP 阶段建议**对白名单/测试者免费**，先验证好玩再谈定价（沿用列王纪的资格框架）。

## 2. 世界设定（童话书 · 由我补全）

### 2.1 这是一本童话书
- 草原是一本旧童话书里的**一页 / 一章**，被某个孩子（**「读者」**）翻看着。**作者**早已写下这片草原的草木与生灵；**读者**每翻一页，世界便流过一段时光——这就是 ×10 时间在故事里的解释：**时间在流动，因为有人在读这个故事。**
- 所有动物**会说话**、有名字、穿戴零碎的小物件（一片叶子做的帽子、一截红绳），懂礼数、讲故事、有传统与歌谣。寓言式的温暖语气。
- 但**童话不等于无害**：狐狸真的会吃掉兔子，寒冬真的会饿死田鼠。可爱的口吻下是**真实的生死**——这份反差是世界的灵魂。
- **昼夜 = 读者的作息**：白天读者在读（明亮、热闹），夜晚读者睡去（黑暗、危险，夜行猎手出没）。**四季 = 故事的章节推进**。

### 2.2 一座岛那么大的草原（地图 · 小而完整）
封闭、可走遍、彼此可达的 ~6 个地点，构成完整食物链与庇护所：
1. **开阔草场（The Open Meadow）**：核心觅食地，草籽野花丰盛，但无遮蔽——猎手的菜园。
2. **兔窟与地洞（The Warren）**：草食动物的地下庇护，安全但出入口是伏击点。
3. **老橡树林缘（Oak's Edge）**：树木、坚果、鸟巢；松鼠与鸦的领地，攀爬与藏匿之所。
4. **芦苇池塘（The Reed Pond）**：水源，青蛙与水鸟，泥泞与倒影；旱季会渐干。
5. **岩丘乱石（The Stony Knoll）**：制高点与洞穴，蛇与鼬的巢，俯瞰全场。
6. **篱笆与稻草人（The Hedge & Scarecrow）**：世界的**边界**（书页的边缘）——越不过去的篱笆，沉默的稻草人，藏着人类世界留下的奇怪物件（童话感）。

> 地图小到一只兔子半天能从一头走到另一头，但每处都有**不同的食物、庇护、危险与社交**，构成自洽的生态。

### 2.3 童话书的"创世"与隐藏大势
- **轻神话**：作者写下草原，吹一口气，万物开口说话；读者一读，时间开始流。
- **隐藏世界时钟（小而真实）**：
  1. **寒冬将至（季节）**：入秋后食物渐少，冬天来临，弱者饿死、有的冬眠——周期性的生存大考。
  2. **池塘渐干（旱季）**：水源紧张，动物被迫聚到池边——猎手的良机。
  3. **「最后一页」（隐藏·存在主义）**：读者总有读完合上书的一天……临近时世界会有微妙征兆（光线变暗、风停、远处书页翻动的巨响）。这是这个小世界的"终极谜团"，留作长线。

## 4. 核心循环（要验证的"最小可玩闭环"）

```
出生(测试→物种) → 醒来在某地点 → [饥饿驱动] 觅食/捕猎(时间占用) → 进食或挨饿
   �‖ 时刻面临 [被捕食风险]：感知危险→逃/藏/对抗(d100) ‖
   → 社交(结盟/示警/求偶) → 筑巢/囤食/迁徙 → 季节/昼夜变化 → 直到：被吃 / 饿死 / 老死(permadeath)
```

- **饥饿系统**：每只动物有饥饿值，随时间上升；草食靠**觅食**（低风险，开阔地有暴露风险），肉食靠**捕猎**（高风险，须追上并制服猎物——NPC 或**别的玩家**）。长期挨饿→掉体力→死亡。
- **捕食判定（确定性 d100）**：猎手 `潜行/狩猎 + 敏捷` vs 猎物 `感官(察觉) → 逃逸/躲藏`。成功=进食并对猎物造成重伤/死亡；失败=猎物逃脱、猎手白费体力。**被捕=重伤或死亡（permadeath）**。
- **非对称玩法**：肉食玩家**必须吃别的动物**（包括玩家）才能活，少而强、被群防；草食玩家多、弱、靠机警与群体。狐吃兔、鼬钻窟、鸮夜袭——天然的 emergent 戏剧。
- **昼夜/季节**（×10 时钟驱动）：夜里猫头鹰/夜行者占优、视野差；寒冬食物稀缺、刺猬等可**冬眠**（长时间占用，躲过严冬但有风险）。
- **动物社会**：会说话→可结盟、示警、传谣、求偶、结群（兔窟/鹿群/鸦群）；情报差让"谁可信、谁在骗你去送死"成为博弈。
- **繁衍 / 血脉传承（核心功能 · 详见 §4A）**：动物可生育后代；玩家死后**可继承自己的在世幼崽**继续，软化 permadeath、形成跨世代血脉。

## 4A. 繁衍与血脉传承（软化 permadeath · 已定为核心）

让"一生"不会因死亡清零，而是延续成一条**血脉**——这既软化永久死亡，又把"开枝散叶、壮大家族"变成真实的长期目标。

- **生育**：成年动物可求偶、交配（与 NPC 或另一玩家），经一段**孕期/育期**（时间占用 + 心跳推进）产下**一窝幼崽**。草食繁衍快（窝大、周期短），肉食繁衍慢（窝小）——契合生态平衡。
- **遗传**：幼崽**继承父母的物种**（血脉延续同一物种），并继承一部分属性/本能/特性（取父母均值 ± 随机波动，偶有"变异"特性）。
- **幼崽是世界里的 NPC**：你当下只操控一只动物（一号一命）；生下的幼崽作为 NPC 在草原长大——也可能被吃掉、夭折（真实生态，不保证都活）。
- **死后传承（关键）**：当你的动物死亡时——
  - 若有**在世直系后代**：可**继承其中一只**，以它的身份继续。它保留你的**物种与血脉加成**（继承部分属性、家族的巢/囤食/领地、部分社会关系与声望），但作为新个体**有自己的天性**——做一次**简短"成长测试"（约 6~8 题）**塑造它自己，父母特质作为起点加成。
  - 若**无在世后代**：回退为**全新随机动物 + 完整 22 题**（血脉断绝，从头来过）。
- **意义**：① 死亡不再清零，血脉延续；② "繁衍、护幼、让家族壮大"成为可追求的长期目标（越早开枝散叶，死后越有得继承）；③ 形成跨世代的**血脉/家族**叙事（"疾风野兔的第七代"），天然契合童话寓言。
- ❓待定：继承时用"短测试塑造新个性"（推荐）还是"直接沿用父母个性"；血脉加成强度（别强到让老家族碾压新生玩家，需平衡）。

## 5. 数值模型（按动物缩放列王纪分层模型）

**测试播种（22 题决定）**
- **A. 属性（1–100，d100 判定底）**：体力 / 力量 / 敏捷 / 感官 / 机敏 / 魅力（＋运）。
- **B. 本能·性向（领域成长倍率）**：狩猎 / 觅食 / 潜行 / 逃逸 / 社交 / 营造（筑巢/掘洞/储食）。
- **C. 性格（对立轴）**：勇敢↔胆怯 ｜ 好奇↔谨慎 ｜ 合群↔独居 ｜ 温和↔凶猛 ｜ 利他↔利己 ｜ 沉稳↔急躁 ｜ 守序↔野性。
- **D. 食性倾向**：由性格（尤其**温和↔凶猛**）+ 狩猎本能综合，决定肉食/草食/杂食的**亲和**，再叠物种稀有度掷定物种。
- **E. 起始特性（离散）**：飞毛腿 / 夜视 / 尖刺 / 利爪 / 假死 / 过目不忘 / 大胃口 / 体弱…（部分随物种天生）。

**出身/一生**：出生地点、所属群体（窟/群/巢）随机；技能（奔跑/攀爬/挖掘/示警/搏斗…）靠一生练（成长=天赋倍率×年龄阶段）；状态（饥饿/伤病/季节/声望）与关系网（同伴/天敌/配偶）活出来。

**判定法**：d100，`掷骰 ≤ 相关属性 + 技能 − 难度` 成功（沿用 CoC 成功等级）。

### 22 题测试 · 样题（示意 3/22 · 确认维度后写全 22 题）
1. 草丛里传来一声你听不懂的响动，你……？　A 立刻冲过去看（勇敢+、好奇+、凶猛+｜亲和:狐/鼬）／B 僵住、竖起耳朵分辨（谨慎+、感官+｜亲和:兔/鹿）／C 悄悄绕开继续干自己的（独居+、潜行+｜亲和:刺猬/獾）／跳过
2. 发现一小堆没人要的果子，你会……？　A 当场吃光（急躁+、利己+｜亲和:鼠）／B 搬回去囤起来（营造+、沉稳+｜亲和:松鼠/獾）／C 喊同伴一起来分（合群+、利他+｜亲和:兔/鹿）／跳过
3. 一只比你弱小的动物挡了你的路，你……？　A 凶它、把它赶走甚至扑上去（凶猛+、力量+｜亲和:肉食）／B 客气地请它让让（温和+、魅力+｜亲和:草食）／C 不声不响从旁边过去（独居+、谨慎+｜亲和:刺猬）／跳过

## 7. 分阶段落地（先验证闭环）

- **阶段 0 · 创世**：锁定草原 canon（地图、10 物种、生态参数）+ 童话基调画风。
- **阶段 1 · 出生与活世界只读**：22 题测试→出生为某动物→醒来在草原；×10 时钟、昼夜季节、饥饿值、编年史、地图浏览。先让"一只会说话的动物站在草原上、肚子会饿"成立。
