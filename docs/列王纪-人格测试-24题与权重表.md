## 1. 算分与归一化规则（给代码）

1. **属性**：`最终 = 30（童年基线） + 种族修正 + Σ测试点 × 3`，clamp 5–80（童年值偏低，靠一生成长抬升）。
2. **天赋**：`最终 = 种族修正 + Σ测试点`（范围约 0–20）。天赋→**成长倍率** `mult = 1.0 + 最终/20`（0.0→1.0×，20→2.0×）。该领域技能成长速度 = 基础 × mult × 年龄阶段系数。
3. **性格轴**：`轴值 = clamp(Σ该极加分 − Σ反极加分, −100..+100) × 系数`。展示为倾向（如"权诈 +40"）。
4. **起始特性**：累计每个特性的 token；**token ≥ 3** 者入候选，取**最高的 1~2 个**为起始特性（并列随机/按稀有度）。种族天生特性（龙息〔龙裔〕、灰变〔灰触者〕等）由种族**额外自动**附带，不占测试名额。
5. **10 族亲和 → 出身种族**：`亲和[族] = Σ该族加分`；`最终权重[族] = 基础稀有度[族] × (1 + 系数 × 亲和[族])`，归一化到 100% 后**加权随机**掷种族——天性加权、命运随机。
   - **普通族**：系数 0.12，单族封顶 ×2.5；灰触者等极稀有族封顶 ×1.8。
   - **魔族特例**：基准 **5%**，**系数 0.2、封顶 ×8**——让"**答得越像魔族**（冷酷、亵渎神明、不择手段逐力、逆天背道），**越可能生为魔族**"；极端黑暗的作答可把魔族概率推高到约 **30–40%**。魔族亲和铺在下列"黑暗"选项上：Q5B、Q8D、Q9D、Q11C、Q13D、Q18C、Q20A、Q23A、Q24A、Q24D。
6. **跳过**：该题对所有维度**不计分**，但在掷种族/算分时，按"**随机抽取该题某一选项的权重**"补入（即跳过 = 命运替你掷）。这实现"跳过会直接按概率生成"。
7. **家庭/出生地**：种族定后，按该族分布 + 各国/宗族人口权重**纯随机**掷定（不受测试影响）。

> 难度判定法（落地后用）：d100，`掷骰 ≤ 相关属性 + 技能 − 难度修正` 成功（沿用 CoC 成功等级）。

## 4. 结果页 · 性格解读（唯一一次 LLM 调用）

算分后，把 `scores`（属性/天赋/性格倾向）+ 命中特性 + **3~5 条最具代表性的作答**喂给 `main` 模型，生成一段第二人称解读：

- "**你是一个怎样的人**"：综合性格轴与天赋，给一段有画面感、像被看穿的人物速写。
- "**为什么这么判断**"：引用玩家的关键选择作依据（如"在最后一块糖前你选择了让给弟妹，又在断粮的寒夜里分粮给陌生人——你的仁善不是说辞"）。
- 收尾点出 1~2 个**起始特性**与**最擅长的领域**，并暗示命运（即将掷定的种族/出身）将把这份天性放进怎样的躯壳里。
- **一次性**：展示后即锁 `test_locked`，不可重测。

> Prompt 草案（`lib/realm/prompt.ts` → `buildPersonaRevealPrompt`）：
> "你是《列王纪》的命运织者。下面是一名新生灵魂的人格测试结果与关键作答。请写一段 150–250 字的第二人称性格解读：先描摹他是怎样的人，再引用其具体选择佐证，最后点出他的天赋特性与最擅长的领域。语气如古老预言、洞悉人心，不堆砌形容词，要有具体画面。只输出 JSON：{\"verdict\":\"…\",\"strengths\":[\"…\"],\"traits\":[\"…\"]}"
