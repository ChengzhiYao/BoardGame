# 搭建指南 · 你需要做的几步

代码我来写，但有几步必须你亲自做——因为涉及你的账号和付款，我替不了。每步都很短，照着做即可。全程不需要你写代码。

---

## 这套东西要花钱吗？

- **搭建 / 代码**：免费（我全写）。
- **托管**：Vercel + Supabase 免费额度，个人测试够用，超量才付费。
- **AI 守秘人 + 图片生成**：**要花钱**，按使用量计费。这是绕不开的真实成本，钱付在你自己的 API key 上。所以代码里做了「图片每局 5–10 张预算」和「成本统计/熔断」来控制开销。

> 等型号定了，我按当时最新单价帮你算「一局大概多少钱」。

## 第一步：装环境（一次性，约 10 分钟）

1. 装 **Node.js**（官网下 LTS 版）。
2. 装 **VS Code**（写不写代码都建议用它，方便填配置、跑命令）。
3. 这个 `coc-trpg` 文件夹就是项目，用 VS Code 打开它。

## 第二步：建 Supabase 项目（数据库 + 登录 + 实时，约 10 分钟）

1. 去 supabase.com 注册，**New project**（区域选离你近的，记下数据库密码）。
2. 左侧 **SQL Editor** → 新建查询 → 把 `supabase/schema.sql` 整个文件内容粘进去 → **Run**。一次建好所有表、权限、防篡改规则。
3. 左侧 **Settings → API**，记下三个值（下一步要填）：
   - `Project URL`
   - `anon public` key
   - `service_role` key（**机密，别外传**）

## 第三步：拿大模型 API key（这步会开始计费）

你选了「Claude 和 GPT 都接、可切换」，所以两家各拿一个：

1. **Claude**：去 Anthropic 控制台（console.anthropic.com）→ 绑支付方式 → 创建 API key（`sk-ant-...`）。
2. **GPT**：去 OpenAI 平台（platform.openai.com）→ 绑支付方式 → 创建 API key（`sk-...`）。建议先充个小额度试水。
3. 图片生成可先复用 OpenAI 的 key。

> 拿到 key 先别贴给任何人，包括聊天框。只填进下一步的本地配置文件。

## 第四步：填配置

1. 把项目里的 `.env.example` 复制一份，改名为 `.env.local`。
2. 按里面的注释，把第二、三步拿到的值一一填进去。
3. `LLM_MAIN_MODEL` / `LLM_AUX_MODEL` 填**官网当前最新的型号名**（强模型当 KP 大脑、便宜模型做摘要省钱）——这块你定不了就把型号发我，我帮你选。

## 第五步：本地跑起来

在 VS Code 的终端里依次运行：

```
npm install
npm run dev
```

浏览器打开 `http://localhost:3000`，能看到落地页就说明地基通了。

## 第六步：部署上线（要联机给朋友玩时再做）

1. 把项目推到 GitHub。
2. 去 vercel.com 用 GitHub 登录 → Import 这个仓库。
3. 在 Vercel 的环境变量里，把 `.env.local` 里的值再填一遍（线上用）。
4. Deploy。拿到正式网址后，把 `NEXT_PUBLIC_SITE_URL` 改成它（邀请链接会用到）。
